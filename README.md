<H1>My Academy Portfolio Project</H1>
<hr>

MVC5 teknolojileri kullanılarak geliştirilen bir portfolyo uygulamasıdır.
<hr>


Kullanılan Teknolojiler:
* ASP.NET
* MVC5
* DB First
* Entity Framework
* SQL Server
